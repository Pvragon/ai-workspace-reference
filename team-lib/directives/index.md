# directives

## Purpose
Store team-level instructions, behavioral rules, and operational guidelines for AI agents.

## What Belongs Here
- Team behavioral directives
- Project-specific instruction sets
- Compliance and safety rules
- Team workflow guidelines

## What Does NOT Belong Here
- Personal directives (use `my-lib/`)
- Persona definitions (use `personas/`)
- Runtime data or logs

## File Naming Conventions
- `<scope>-<name>.md` (e.g., `team-coding-standards.md`, `project-example-guidelines.md`)

## How Agents Should Use This Folder
Agents should load team directives for any pvragon-related task execution.
