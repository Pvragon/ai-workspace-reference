# logs

## Purpose
Team execution logs and audit trails.

## What Belongs Here
- Team execution logs
- Audit records
- Shared activity history

## What Does NOT Belong Here
- Personal logs (use `my-lib/`)
- Active runtime data

## File Naming Conventions
- `YYYY-MM-DD-<execution-id>.log`

## How Agents Should Use This Folder
Write logs for team debugging and auditing.
