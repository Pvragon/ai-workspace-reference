# _admin

## Purpose
Administrative scripts and tooling for workspace management.

## What Belongs Here
- Bootstrap scripts
- Maintenance scripts
- Administrative tools
- Workspace management utilities

## What Does NOT Belong Here
- Project source code
- AI agent resources
- Runtime data

## File Naming Conventions
- `<action>_<target>.sh` for scripts (e.g., `bootstrap_local.sh`)
- Use snake_case for script names

## How Agents Should Use This Folder
Reference for workspace administration tasks.
