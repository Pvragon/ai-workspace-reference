# global

## Purpose
Essential context that should always be available to agents regardless of task.

## What Belongs Here
- Core domain knowledge
- Frequently referenced information
- Universal conventions and standards

## What Does NOT Belong Here
- Large documents (use `indexed/`)
- Task-specific context
- Temporary information

## File Naming Conventions
- Short, descriptive names
- Keep files small for fast loading

## How Agents Should Use This Folder
This context is loaded automatically at agent initialization.
