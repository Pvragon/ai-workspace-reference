# Layer 2 — Personal Extension Library

You are in the **personal extension library**. This is a privately versioned workspace for developing and testing automations before graduating them to the team library.

## Operating Mode: Automation

- Follow DOE patterns: directives define intent, executions do the work
- Push here while working — this is the sandbox
- Use `runtime/` for temp files and deliverables
- Use `backlog/` for planned/paused initiatives
- Graduate mature tools to team-lib via PR
